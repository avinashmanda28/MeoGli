# MeoGli Frontend

Next.js + Tailwind CSS frontend for MeoGli AI video generator.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Clean dark UI with blue accents
- Real-time progress tracking
- IdeaFlow guided assistant
- Video preview and download
- Responsive design
